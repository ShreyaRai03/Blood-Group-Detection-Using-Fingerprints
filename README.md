#  Blood Group Detection Using Fingerprint Analysis

##  Overview
This project uses Machine Learning and Image Processing techniques to predict a person’s blood group based on fingerprint patterns. The system extracts features from fingerprint images and applies classification algorithms to identify blood group categories.

---

##  Objective
To explore biometric-based prediction by analyzing fingerprint structures and building a machine learning model that classifies blood groups accurately.

---

##  Tech Stack
- Python  
- OpenCV  
- NumPy  
- Scikit-learn  
- Machine Learning  

---

##  Workflow
1. Load fingerprint dataset  
2. Preprocess images (grayscale conversion, noise removal)  
3. Extract meaningful features from fingerprint patterns  
4. Train machine learning model  
5. Predict blood group from input fingerprint image  

---

##  Machine Learning Model
- Classification-based ML algorithms (e.g., SVM / Random Forest depending on implementation)

---
